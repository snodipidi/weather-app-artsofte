# Weather-App

Приложение "Погода": https://weather-app-artsofte.netlify.app/

Доработанный мной макет Figma: (https://www.figma.com/design/fkbiETaN3ysuCvEGKjocEa/Untitled?node-id=0-1&t=kl9RjjjAZq4RkAzq-1)

## Возможности
- Валидация широты/долготы
- Загрузка погоды через `fetch` (Open‑Meteo): температура, ощущается как, влажность, ветер, код погоды, локальная тайм‑зона и время в локации
- Несколько виджетов
- Разные иконки под погоду
- Карта локации в карточке (static-maps.yandex.ru) с пином по координатам

## API
- Погода: [Open‑Meteo](https://open-meteo.com/)
- Карты: [static-maps.yandex.ru](https://static-maps.yandex.ru/)

## Как пользоваться
1. Введите широту и долготу
2. Нажмите «Показать погоду»
3. Температура, локальное время, «ощущается как», ветер, влажность и карта появятся в карточке
4. Чтобы удалить карточку, нажмите «×»

Ошибки ввода и недоступность API показываются под формой или внутри карточки.
